Next.JS 16 - Note Taking App
======================================================================

A small CRUD app build with Next.JS 16  
(& TS + Pocketbase)


## Installation

Wanna try it out yourself ? 

Pull the code and, from the root of the directory, install dependencies,
```sh
npm install 
```

run the local pocketbase server
```sh
./pocketbase serve 
```

& run the dev build
```sh
npm run dev 
```

VoilÃ  ! You can access the app at http://localhost:3000.
